# Junior Fullstack Developer Challenge - Nimble Gravity

## Requisitos

- Node.js instalado
- Un archivo `.env` en la raiz del proyecto con la variable de entorno `VITE_API_BASE_URL` con la url base de la api

## Instalacion

Instalar las dependencias del proyecto:

```
npm install

```

Las herramientas utilizadas son:
- React
- Vite
- Tailwind CSS

## Iniciar el servidor de desarrollo

```
npm run dev
```

El servidor se iniciara en `http://localhost:5173`.
